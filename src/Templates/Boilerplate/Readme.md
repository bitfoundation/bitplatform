# bit Boilerplate

Free, open-source, full-stack, cross-platform, cloud-ready, and battle-tested project template for C# and Blazor.

## Features

- **GenAI & RAG** integration
- **Aspire** for cloud-native development
- **Ultra-fast response** with data caching
- **Identity**: Passwordless, 2FA, SSO, and more

## Demos

Try it out on [Google Play, App Store, Web, and Windows](https://bitplatform.dev/demos).
