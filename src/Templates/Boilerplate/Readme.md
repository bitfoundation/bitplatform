
# bit Boilerplate

Free, open-source, full-stack, cross-platform, cloud-ready, and battle-tested project template for those who love C# & Blazor ❤️

## Features

###  Modern Web Apps
- **Public facing Web Apps** Pre-render enabled, Ultra-fast, SEO-Friendly Websites using Blazor WebAssembly! [Demo](https://bitplatform.dev)
- **AdminPanels** Realtime Dashboard, DataGrid etc. [Demo](https://adminpanel.bitplatform.dev)
- **Offline-capable PWA** with client-side database and sync feature. [Demo](https://todo-offline.bitplatform.cc/offline-todo)
- **Blazor WebAssembly standalone** For Azure Static Pages, GitHub Pages, Cloudflare Pages etc. [Demo](https://adminpanel.bitplatform.cc)

### Cross-Platform Native Apps
Generate production-ready **Android**, **iOS**, **Windows**, and **macOS** applications from a single codebase using .NET MAUI Blazor Hybrid. [Demos on Google Play, Apple Store & Windows EXE](https://bitplatform.dev/demos#adminpanel)

### Client-Side Offline Database
Full **Entity Framework Core + SQLite** support on the client side with data synchronization across Web, Android, iOS, Windows, and macOS platforms.

### Localization & Globalization
Complete **multi-language support** with **RTL** (Right-to-Left) and **LTR** (Left-to-Right) layout capabilities built-in.

### Advanced Caching
Comprehensive **Data and Response caching**:
- In-memory caching
- Redis caching
- Hybrid (memory + Redis) caching
- Output caching
- CDN Edge caching (Cloudflare integration)
- Browser & Client device caching

### Observability & Monitoring
- **OpenTelemetry** for distributed tracing and metrics
- **Sentry** integration for error tracking
- **Azure Application Insights** for comprehensive monitoring

### Identity & Security
- Passwordless authentication
- Two-factor authentication (2FA)
- Single Sign-On (SSO)
- And more...

### GenAI & RAG
- AI Chatbot integration
- Powerful vectorized database search using embeddings

### Modern Stack
- **ASP.NET Core 10**, **C# 14**, **.NET Aspire**
- **Entity Framework Core** with multiple database support (SQL Server, PostgreSQL, MySQL, SQLite)
- Flexible file storage (Local, S3, Azure Blob Storage)
- CI/CD pipelines for GitHub Actions and Azure DevOps

### ✨ And more...
All features can be **selectively chosen during project creation** to match your specific requirements!
