# bit Boilerplate

Free, open-source, full-stack, cross-platform, cloud-ready, and battle-tested project template for those who love C# & Blazor ❤️

## Features

- **Modern** ASP.NET Core 10, Azure Cloud, C# 14, Aspire
- **Ultra-fast** Response Caching on Origin, Edge CDN and Client, Data Caching using Hybrid Redis/In-Memory
- **Identity**: Passwordless, 2FA, SSO
- **GenAI & RAG** AI Chatbot, powerful vectorized database search
- **AdminPanel** Dashboard, DataGrid
- And a lot more!
